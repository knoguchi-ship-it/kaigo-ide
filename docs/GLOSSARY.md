# 用語集 (Glossary)

## 介護保険・業務用語

| 用語 | 読み | 説明 |
|------|------|------|
| 居宅介護支援 | きょたくかいごしえん | 要介護者が自宅で介護サービスを利用するためのケアプラン作成・調整を行うサービス |
| 介護支援専門員 | かいごしえんせんもんいん | ケアマネジャー。ケアプラン作成、サービス調整を行う専門職 |
| ケアプラン | - | 居宅サービス計画書。利用者の状態・目標・サービス内容をまとめた計画書 |
| 第1表 | だいいちひょう | 居宅サービス計画書（1）。利用者の意向、総合的な援助の方針等 |
| 第2表 | だいにひょう | 居宅サービス計画書（2）。生活全般の解決すべき課題、目標、サービス内容 |
| 第3表 | だいさんひょう | 週間サービス計画表。1週間のサービス利用スケジュール |
| 第4表 | だいよんひょう | サービス担当者会議の要点。会議の出席者、検討内容、結論等の記録 |
| 第5表 | だいごひょう | 居宅介護支援経過。日々の支援内容・連絡・判断等の経過記録 |
| 第6表 | だいろくひょう | サービス利用票。月間のサービス利用予定と実績 |
| 第7表 | だいななひょう | サービス利用票別表。単位数・費用の明細 |
| サービス担当者会議 | - | ケアマネジャーがサービス事業者、利用者・家族を集めて行う会議 |
| アセスメント | - | 利用者の状態・課題を分析・評価するプロセス |
| モニタリング | - | ケアプランの実施状況を定期的に確認・評価するプロセス |
| 要介護度 | ようかいごど | 介護の必要度を示す区分（要支援1〜2、要介護1〜5） |
| 被保険者番号 | ひほけんしゃばんごう | 介護保険の被保険者に付与される10桁の番号 |
| 保険者番号 | ほけんしゃばんごう | 市区町村に付与される6桁の番号 |
| LIFE | ライフ | 科学的介護情報システム。介護サービスのデータを収集・分析するシステム |
| 国保連 | こくほれん | 国民健康保険団体連合会。介護報酬の審査・支払いを行う |
| 給付管理 | きゅうふかんり | 利用者ごとのサービス利用単位数の管理 |

## 技術用語

| 用語 | 説明 |
|------|------|
| SPA | Single Page Application。ページ遷移なしで動作するWebアプリケーション |
| PWA | Progressive Web App。ネイティブアプリのような機能を持つWebアプリ |
| API | Application Programming Interface。システム間のデータ連携インターフェース |
| REST | Representational State Transfer。HTTPベースのAPI設計スタイル |
| JWT | JSON Web Token。認証情報を含むトークン形式 |
| ORM | Object-Relational Mapping。オブジェクトとDBを橋渡しするツール |
| モノレポ | Monorepo。複数プロジェクトを単一リポジトリで管理する構成 |
| マルチテナント | 単一システムで複数の組織（テナント）のデータを分離して管理する設計 |
