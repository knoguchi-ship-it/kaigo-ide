## 1. プロジェクト概要 (Project Overview)

### 1.1 背景 (Background)
居宅介護支援事業所（ケアマネジャー）は、日々の支援経過記録、担当者会議の記録、ケアプラン作成など多岐にわたる書類作成業務を行っている。現状、多くの事業所では紙ベースまたは複数のツールを併用しており、業務効率化が課題となっている。

2024年の介護報酬改定により、ケアプランデータ連携システムの活用が居宅介護支援費（Ⅱ）の要件に組み込まれるなど、ICT化の推進が加速している。

### 1.2 目的 (Purpose)
本プロジェクトは、介護支援専門員の業務効率化を実現するブラウザベースの総合業務支援システム「KaigoIDE」を開発する。

### 1.3 目標 (Objectives)
| ID | 目標 | 測定指標 |
|----|------|----------|
| OBJ-1 | 記録作成時間の削減 | 従来比30%削減 |
| OBJ-2 | ペーパーレス化の推進 | 紙帳票80%削減 |
| OBJ-3 | マルチデバイス対応 | PC/タブレット/スマホで同等機能 |
| OBJ-4 | 標準仕様への準拠 | ケアプランデータ連携システム対応 |
